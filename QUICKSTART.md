# Быстрый старт

## Запуск проекта

### Вариант 1: Командная строка
```bash
cd CateringIS
dotnet restore
dotnet run
```

### Вариант 2: Visual Studio
1. Открыть файл `CateringIS.csproj`
2. Нажать F5 или кнопку "Запуск"

---

## Где хранятся данные?

Все данные сохраняются автоматически в папке **Data/** рядом с программой:

**При разработке:**
```
CateringIS/bin/Debug/net8.0-windows/Data/
  ├── suppliers.json, products.json и т.д.
  └── Photos/ (фотографии блюд)
```

**После публикации:**
```
<папка_с_программой>/Data/
  ├── suppliers.json, products.json и т.д.
  └── Photos/ (фотографии блюд)
```

**Удобно:** все данные в одном месте, легко найти, скопировать или сделать резервную копию!

## Первые шаги

1. **Добавьте поставщика** (раздел "Поставщики")
2. **Добавьте ингредиенты** (раздел "Склад")
3. **Создайте группы блюд** (раздел "Группы блюд") — уже есть стандартные
4. **Добавьте блюда** с фото и рецептурой (раздел "Меню")
5. **Зарегистрируйте приход товара** — остатки обновятся автоматически
6. **Создайте заказ** — остатки спишутся автоматически
7. **Учитывайте продажи** (раздел "Учёт продаж")
